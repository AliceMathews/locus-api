# Locus

## Getting Started

1. Create the `.env` by using `.env.example` as a reference: `cp .env.example .env`
2. Install dependencies: `npm i`
3. To reset database: `npm run db:reset`
4. Run the server: `npm run local`
5. Visit `http://localhost:8080/`

## Dependencies

- Node 10.x or above
- NPM 5.x or above
- PG 6.x
